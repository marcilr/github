From creation of syntax repo on github:

Quick setup - if you've done this kind of thing before
  or HTTPS   https://github.com/marcilr/git-github-syntax.git
     SSH     git@github.com:marcilr/git-github-syntax.git

We recommend every repository include a README, LICENSE, and .gitignore.
  
  or create a new repository on the command line



echo "# git-github-syntax" >> README.md
git init
git add README.md
git commit -m "first commit"
git remote add origin git@github.com:marcilr/git-github-syntax.git
git push -u origin master
...or push an existing repository from the command line

git remote add origin git@github.com:marcilr/git-github-syntax.git
git push -u origin master
...or import code from another repository

You can initialize this repository with code from a Subversion, Mercurial,
or TFS project.
