List remotes

To list all remotes use 'git remote -v'

-- git-scm.com/



Example
=======
covellite:~/git/github/remotes/syntax$ git remote -v
origin  ssh://git@github.com/marcilr/git-github-remotes-syntax (fetch)
origin  ssh://git@github.com/marcilr/git-github-remotes-syntax (push)
covellite:~/git/github/remotes/syntax$
